# Bakery Market Insights Tool

A Python-based utility designed for market research within niche bakery and small-business communities on Reddit.

## Purpose

This tool helps identify recurring business challenges (such as product costing, pricing strategies, and operational hurdles) to inform the development of more effective, user-centered software tools for entrepreneurs.

## Compliance

This script is built using the PRAW library and adheres to Reddit's API guidelines. It is strictly a read-only research utility. No automated posting, messaging, or spamming functionality is included.

## Setup

1. Clone the repository.
2. Install dependencies: `pip install praw`
3. Configure your environment variables for `client_id` and `client_secret`.
